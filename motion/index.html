Worked with Prismport (A Webflow Agency) to build an interactive visualisation for the working of the newscatcher api. The goal was to teach a non-programmer about the complicated steps in an easy way. Being a developer also helped visualise the provided steps.

Refresh

A personal hobby project - an app that helps you follow the 20-20-20 rule. It sends custom notifications as an overlay with a rive animation. Built with flutter, I didn't restrict myself for any crazy idea, from a mood selector to custom icon transitons in-between pages.

I believe as much as the functional value, the perceived value is as important. Not only full-filling the users needs but also making the experience satisfying. Also, making sure not to implement every crazy idea that comes to mind, that may look cool but compromise the user experience.
